Off-Road Route Planner (ORP)

A full-stack route optimization system for off-road terrain using custom A* pathfinding. The system processes elevation data from Google APIs to generate terrain-aware paths that avoid slopes exceeding 30% incline. Features interactive mapping, real-time elevation analysis, and multi-waypoint planning for areas without roads or trails.
